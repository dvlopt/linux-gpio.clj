## [2.0.0] - 2021-03-24

### Changed

- Pluralization convention:
    - defaults     -> default+
    - clear-lines  -> clear-line+
    - get-lines    -> get-line+
    - set-lines    -> set-line+
    - toggle-lines -> toggle-line+
- Namespace parameters under "gpio"
- Relicensed under MPL 2.0
- Renamed group id from "dvlopt" to "io.helins"
